## Usage

The big serif quote mark scales in first, the quote text fades in, then the avatar + attribution line slides up. Use `https://unavatar.io/twitter/<handle>` if you want avatars resolved straight from a Twitter handle.

Drop multiple testimonials sequentially in your project for a "what users are saying" section.
