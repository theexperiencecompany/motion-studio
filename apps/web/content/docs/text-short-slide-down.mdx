## Usage

Each new word drops in from above into its own line and pushes the existing stack downward until a centered three-line composition locks in place. Drop it in as a chapter card, hero opener, or as a beat between scenes — the same headline + subtitle props power the whole text-animation family, so swapping motions takes one field change.
