## Usage

Font Hook keeps the scene intentionally simple: centered text on a flat
background, with the font family swapping every few frames. Use the `headline`
field for the main hook. The optional `subtitle` field renders as smaller plain
text below it when provided.

The Style panel still applies. Background color, text color, and font family can
be overridden, and the selected font is inserted into the rotation. There are no
gradient layers, accent bars, shadows, or decorative effects.
