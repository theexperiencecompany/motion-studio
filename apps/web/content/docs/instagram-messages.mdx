## Usage

Instagram DM variant of the messages template. `side: "right"` is the gradient sent bubble (Instagram's signature purple → pink → orange), `side: "left"` is the grey received bubble. The header shows the contact name with an `Active now` indicator and the gradient story ring around the avatar.

Same `ChatMessage` shape as the iMessage and WhatsApp components — `text`, `side`, `typingFrames`, `delay`. Bubbles spring in, the typing dots show until `typingFrames` elapses, then the bubble pops in. Older messages slide upward when newer ones arrive.
