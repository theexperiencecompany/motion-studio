## Usage

A multiplier component. Pick any other composition from the library and it renders inside a realistic iPhone mockup — Dynamic Island or classic notch — with a spring drop-in entrance.

The inner composition uses its own default props and renders at its native dimensions, scaled to fill the phone screen with `cover` fit. Pair this with chat compositions like `MessageBubbles` or `WhatsAppMessages` for a phone-on-screen reaction shot, or wrap a `BrowserScroll` recording to mockup a mobile app demo.

The composition is 1080×1920 (9:16) so it slots straight into TikTok, Reels, and Shorts pipelines.
