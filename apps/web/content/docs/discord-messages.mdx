## Usage

Discord-channel layout in the standard dark theme. Round avatars, colored usernames per side (blurple for `left`, red for `right`), Discord-style "Today at HH:MM" timestamp, and a typing line at the bottom while a message is being typed.

`contactName` becomes the `#channel-name` in the header. Per-message `side` controls who the message is from — `left` uses the channel/contact name, `right` uses `you`.
